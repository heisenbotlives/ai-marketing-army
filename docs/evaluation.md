# Evaluation

AI marketing systems need quality gates. Speed is only useful if the output is accurate, useful, and on strategy.

## Output scorecard

Each output can be scored from 1 to 5 on:

- **Accuracy:** Are claims supported by evidence?
- **Clarity:** Is it easy to understand?
- **Usefulness:** Does it help the intended audience?
- **Strategic fit:** Does it support the positioning or campaign goal?
- **Originality:** Does it avoid generic AI-sounding copy?
- **Tone:** Does it sound appropriate for the brand/audience?
- **Actionability:** Is there a clear next step or decision?
- **Risk:** Are there privacy, compliance, or reputational concerns?

## Research quality checks

- Source links included
- Date/context captured
- Claims separated from interpretation
- Confidence level stated
- Open questions listed

## Reviewer decision options

- **Approve:** Safe to use after human review.
- **Revise:** Useful direction, but needs changes.
- **Reject:** Not accurate, useful, or strategically relevant.
- **Escalate:** Needs human/domain expert judgement.
