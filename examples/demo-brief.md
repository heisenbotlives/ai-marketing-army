# Demo Brief: AI Agents for Marketing Teams

## Context

This is a sanitised example brief for the portfolio repo. It does not use private employer notes or proprietary research.

## Audience

Marketing leaders and operators who are curious about AI agents but unsure how to apply them safely.

## Goal

Explain that the value of AI agents is not just faster content production. The deeper value is repeatable research, synthesis, review, and learning loops.

## Key message

AI agents become useful when they are designed as part of a marketing operating system, not when they are treated as isolated chatbot prompts.

## Suggested outputs

- LinkedIn post explaining the system view
- Blog outline on agentic marketing workflows
- Diagram showing research-to-content pipeline
