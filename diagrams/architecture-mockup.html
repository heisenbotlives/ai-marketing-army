AI Marketing Army Architecture

Mockup: six AI marketing agents coordinate through a shared Kanban board, shared knowledge base, MCP connectors, and one human approval gate.

What this shows

  • • Six agents coordinate in Hermes Agent
  • • Orchestrator is one agent, not a separate layer
  • • Kanban is the shared work-state system
  • • Agents create, claim, and update cards

Knowledge base

  • • Brand & Reference: Obsidian / GitHub
  • • Operational Data: Google Drive
  • • Skills: GitHub / Hermes markdown skills
  • • No duplicated knowledge-source boxes

Governance

  • • Human Approval Gate appears once
  • • Customer-facing output passes through it
  • • Safety is process governance, not a KB source
  • • Diagram avoids overclaiming full autonomy
